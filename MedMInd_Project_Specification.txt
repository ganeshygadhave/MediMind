
MEDREM - AI POWERED MEDICATION REMINDER & HEALTH ASSISTANT
==========================================================

PROJECT OVERVIEW
----------------
MedRem is a secure Android healthcare application designed to help users manage medications,
receive medication reminders, upload prescriptions and medical reports, track medication
consistency, and interact with a context-aware AI health assistant.

-------------

WORKFLOWS
---------

Prescription Upload:
Upload -> Cloudinary -> Gemini -> Summary -> MongoDB

Medicine Extraction:
Upload -> Gemini -> Detect Medicines -> Approve/Edit -> Save Medication

Chatbot:
Question -> JWT Verify -> Fetch Profile + Medications + Summaries
-> Gemini -> Response

Reminder:
Medication -> Schedule -> FCM Notification
-> Taken / Skip / Remind Later

SECURITY
--------
- JWT Authentication
- bcrypt Password Hashing
- HTTPS
- User-level Authorization
- Input Validation
- Secure Cloud Storage

AI RULES
--------
- Inform, summarize and assist
- Never diagnose
- Never prescribe medication
- Never recommend stopping medication
- Never replace healthcare professionals
- Calm, safe and supportive tone
